# KNN Handwritten Digit Classification

## Objective
To classify handwritten digits using K-Nearest Neighbors and understand distance-based learning.

## Dataset
- Sklearn Digits Dataset (8×8 images)

## Tools Used
- Python
- Scikit-learn
- Matplotlib
- Seaborn

## Workflow
1. Load and inspect dataset
2. Visualize sample digits
3. Train-test split
4. Feature scaling using StandardScaler
5. Train KNN with multiple K values
6. Select best K using accuracy
7. Generate confusion matrix
8. Visualize predictions
9. Save all outputs automatically

## Results
- Best K selected based on accuracy
- Accuracy vs K graph
- Confusion matrix visualization
- Final predictions displayed
